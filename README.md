# Customer Churn Analysis

## Objective
Analyze customer behavior and identify factors contributing to subscription cancellations.

## Tools
- Python
- Pandas
- Matplotlib

## Analysis Performed
- Churn distribution
- Monthly charges analysis
- Contract type analysis
- Tenure analysis
- Payment method analysis

## Key Findings
- Month-to-month customers churn more.
- Higher charges increase churn risk.
- Short-tenure customers are more likely to leave.

## Recommendations
- Encourage long-term subscriptions.
- Improve onboarding.
- Launch retention campaigns.
